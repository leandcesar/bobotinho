O bobotinho é um bot de variedades voltado para o entretenimento de chats offstream da Twitch brasileira. Ele conta com diversas funcionalidades para que a experiência dos usuários que permanecem no chat mesmo quando a stream está offline seja mais divertida.

Caso queira colaborar, recomendo fortemente que entre no Discord para ter acesso a organização do projeto e criação das ideias.
